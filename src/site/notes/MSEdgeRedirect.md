📄众所周知，科技巨头们因为欧盟数字市场法案，所以都会做一些妥协。

比如欧洲的 Windows 拥有一些其他地区没有的功能：

▫️卸载 Edge 浏览器
▫️可以更换 Windows 默认搜索引擎、浏览器
▫️关闭 Windows 开始菜单、小部件广告等

而使用 MSEdgeRedirect ，你也可以为你的 Windows 解禁这些功能了，原理是修改系统内置的区域代码为欧洲地区
